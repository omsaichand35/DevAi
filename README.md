# DevAi

A collaborative code editor with real-time file synchronization.

## Components
- **DevAi**: Windows Forms Code Editor.
- **collab-dotnet-server**: SignalR server for real-time collaboration.
- **collab-dotnet-client**: Console client for file synchronization.

## How to Run
1. Run `run_demo.ps1` in PowerShell to start the full environment (Server + 2 Clients + 2 Editors).
